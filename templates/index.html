How this works:

  • User makes a POST request to the API.
  • Sends an Image with a key 'Image' in the files dictionary

Routes:

  • Base URL: https://emoplay.herokuapp.com
  • Scrape Request: https://emoplay.herokuapp.com/
  • Request Method: POST
  • Scheme:
            
#PYTHON EXAMPLE

            
            import requests
            url = 'https://emoplay.herokuapp.com'
            response = requests.post(url, files={'image': open('local_path_to_your_image_file', 'rb')})
